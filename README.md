# Solaren Engineering

ابزار موبایلی طراحی و برآورد سیستم‌های خورشیدی.

## اجرای مستقیم

فایل `index.html` را در مرورگر باز کنید. محاسبات و ذخیره محلی کار می‌کنند، اما نصب PWA در حالت `file://` فعال نیست.

## اجرای کامل و نصب PWA

پوشه را روی یک میزبان HTTPS قرار دهید یا در کامپیوتر از سرور محلی استفاده کنید:

```bash
python3 -m http.server 8080
```

سپس آدرس زیر را باز کنید:

```text
http://localhost:8080/solaren-single/
```

مرورگر پشتیبانی‌شده گزینه نصب برنامه را نمایش می‌دهد.

## فایل‌ها

- `index.html` — رابط، محاسبات و ذخیره‌سازی
- `manifest.webmanifest` — مشخصات نصب برنامه
- `sw.js` — Cache و اجرای آفلاین
- `icons/` — آیکن‌های نصب

## نکته فنی

خروجی‌های طراحی باید پیش از اجرا با دیتاشیت رسمی تجهیزات، شرایط واقعی محل و مقررات برق تطبیق داده شوند.
